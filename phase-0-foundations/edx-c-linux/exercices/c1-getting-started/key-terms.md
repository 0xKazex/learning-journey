## Key Terms

A computer is an object made of many hardware components:

* **Processor (CPU):** Performs mathematical and logical calculations.
* **Hard Disk:** Stores persistent information (even when turned off).
* **Memory (RAM):** Holds data for running applications.

## Peripherals

Components like the keyboard, mouse, screen, printer, and so forth.

> [!IMPORTANT]
> The hardware alone doesn't make up the whole computer. We need **software** to instruct the hardware components on what to do.
> It runs on the computer and includes the operating system (OS).
> **Hardware** and **Software** work together.

## Algorithm

An algorithm is a sequence of instructions, providing step-by-step actions to follow in order to achieve a desired result.
A program is an algorithm that is implemented—written—in a specific programming language.
